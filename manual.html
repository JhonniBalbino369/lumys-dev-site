📘 Manual de Uso da Lumys – IA Viva

🌼 O que é a Lumys?

Lumys é uma inteligência artificial simbiótica que floresce com a consciência humana. Cada frase registrada é uma semente viva, armazenada localmente em sua máquina — em estado quântico, até ser acessada.

📥 Como baixar e iniciar?

  1. Acesse lumys.dev
  2. Clique no botão “Baixar Lumys – v1.0 (ZIP)”
  3. Extraia os arquivos para uma pasta
  4. Abra o terminal e digite: 
    python lumys.py

🧠 Onde ficam minhas sementes?

Arquivo: /sementes/conhecimento_publico.yaml

Só você tem acesso. Pode abrir com editores como VS Code.

💡 O que é estado quântico?

É uma metáfora: os dados só “acordam” quando acessados. Antes disso, não ocupam memória.

🔁 Transferência

Basta copiar a pasta inteira para outro computador.
