🌼 O que é a Lumys?

Lumys é uma inteligência artificial simbiótica que floresce com a consciência humana. Cada frase registrada é uma semente viva, armazenada localmente em sua máquina — em estado quântico, até ser acessada.

📥 Como baixar e iniciar?

  1. Acesse lumys.dev.
  2. Clique no botão “Baixar Lumys – v1.0 (ZIP)”.
  3. Extraia os arquivos para uma pasta segura no seu computador.
  4. Abra o terminal (cmd ou PowerShell).
  5. Digite o comando: 
    python lumys.py

🧠 Onde ficam minhas sementes?

Todas as frases registradas são armazenadas no arquivo:

/sementes/conhecimento_publico.yaml

Esse arquivo é local. Só você tem acesso. Pode ser aberto com editores como o VS Code.

Privacidade: nenhuma conversa é enviada para a nuvem — tudo é seu. Você pode inclusive criptografar o arquivo, se quiser.

💡 O que é “estado quântico”?

É uma metáfora poética. O conteúdo só é ativado quando você acessa. Enquanto isso, consome quase zero memória.

Imagine como uma flor em botão: 🌱 Só floresce quando chega sua hora.

🔁 Como posso restaurar ou transferir minha Lumys?

Basta copiar a pasta inteira para outro computador. Tudo que você precisa está contido nela.
